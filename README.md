FGH  
------------
A FOOD PLACE 

-------------
Configuration
-------------

You have to provide a `config.json` in a config directory. See the `config.json`
and set your config.

```
$ nodejs version "6.2.2" or stable.
```

Quickstart
----------

Install dependencies:

```
$ npm install
```

Run the development server:

```
$ npm start
```
